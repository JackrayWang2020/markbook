# 0、Flink概述

## 1、Flink核心特性是什么？

### 1.1、批流处理

- 技术含义:
  Flink将批处理视为流处理的特例（有限流），统一了批流引擎。
  - 流处理： 无限数据流（如kafka实时日志）
  - 批处理： 有限数据流（如hdfs上的csv文件）
- 优势：
  - 一套API/引擎处理所有场景
  - 避免Lambda架构（批流双链路）的运维复杂性
